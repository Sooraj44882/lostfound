# Lost & Found

A prototype built for our school to help students track lost and found items more easily. 

## Features

* **Post & Remove Listings**:  Add or delete lost/found items with a title and contact email.
* **Image Uploads**:  Attach a photo to a listing (images are stored using Base64 encoding in ).
* **Search**:  search by item name or email.
* **Filter**:  Filter the board by Lost or Found .
* **Bulletin Board Look**:  Styled to resemble  actual school corkboard.


## Tech Stack

* **HTML** 
* **CSS** 
* **JavaScript** 

## How to Run

1. Clone or download the repo.
2. Open `index.html` in a browser.
